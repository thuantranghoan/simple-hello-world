# Simple Hello World

Một ứng dụng Next.js đơn giản với giao diện đẹp mắt và hiện đại.

## 🎨 Tính năng

- Landing page với gradient background ấn tượng
- Hero section với các call-to-action buttons
- Features section giới thiệu 3 điểm nổi bật
- Statistics section hiển thị các số liệu
- Responsive design với Tailwind CSS
- Animation mượt mà

## Cài đặt

```bash
npm install
```

## Chạy ứng dụng

```bash
# Chạy dev server
npm run dev

# Build cho production
npm run build

# Chạy production build
npm start
```

Mở [http://localhost:3000](http://localhost:3000) để xem kết quả.

## Công nghệ

- [Next.js](https://nextjs.org/) - React framework
- [React](https://reactjs.org/) - UI library
- [TypeScript](https://www.typescriptlang.org/) - Type safety
- [Tailwind CSS](https://tailwindcss.com/) - Utility-first CSS framework
